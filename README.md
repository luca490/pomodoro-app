# ⏱️ Cronômetro Pomodoro em Python

Um aplicativo desktop feito em Python com interface gráfica para controle de tempo no estilo Pomodoro.

## 🚀 Funcionalidades

* ⏱️ 35 minutos de foco (Exercícios)
* 🧠 25 minutos de descanso (Resumo)
* ▶️ Botão de iniciar
* ⏸️ Pausar / continuar
* 🔄 Resetar ciclo
* 📊 Barra de progresso
* 🔊 Sons diferentes para cada fase

## 🖥️ Tecnologias

* Python
* Tkinter (interface gráfica)

## ▶️ Como executar

```bash
python app.py
```
